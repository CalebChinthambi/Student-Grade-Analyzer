# Student Grade Analyzer

## Overview
A clean, modular Python script designed to automate the process of grading student assessments. This project demonstrates fundamental programming concepts such as function definition, list processing, and error handling.

## Logic & Features
* **Modular Design:** Separate functions for calculating averages and determining grade boundaries for better maintainability.
* **Input Handling:** Processes multiple scores simultaneously via comma-separated user input.
* **Formatted Output:** Provides a clear breakdown of individual scores alongside final calculated results.

## Technical Skills Demonstrated
* Python Fundamentals
* Data Validation & Error Handling (Try-Except blocks)
* List Comprehension
* String Formatting

## Usage
1. Run the script: `python grade_analyzer.py`
2. Enter scores when prompted (e.g., 85, 92, 78).
3. View the generated grade report.
